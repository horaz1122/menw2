💻 بەشی یەکەم: چەمکەکانی CSS

  • CSS زمانێکە دیزاین کراوە بە شێوەیەکی ئاسان دەتوانرێت پێی جێبەجێ بکرێت بۆ دروستکردنی پەیجێک یان وێبپەیجێک.
  • CSS ڕێگەمان پێدەدات ستایلێک جێبەجێ بکەین لەسەر فایلێکی HTML.
  • Declaration بەکاردێت بۆ ناساندنی پرۆپێرتییەکە.
  • Selector واتا ئیلیمێنتێک بۆئەوەی ستایلی بدەیتێ.
  • Property واتا تایبەتمەندییەک دەدەین بە ستایلێک.
  • Value نرخێکە دەدرێت بە پرۆپێرتییەک بۆ ئیلیمێنتێکی CSS کە دیاری دەکرێت.
  • کۆدی CSS لەناو تاگی style کەمان لەناو head ییدا دەنووسرێت.

بۆچی CSS بەکاردەهێنرێت؟ (خاڵ بەندی)

  • 1- Saves Time
  • 2- Easy Maintenance
  • 3- Search Engine Friendly
  • 4- Superior Styles
  • 5- Offline Browsing

کورتکراوەی CSS: Cascading Style Sheets

🎯 بەشی دووەم: جۆرەکانی Selectors لە CSS

CSS Selectors: ستایلێک دەدەین بە ئیلیمێنتێک لە HTML وە ڕێگە ئەدات زیادی بکەین بۆ تاگ یان هەر شتێکی تر.

  • Simple Selectors (سێلێکتەری سادە): ئەوانە دەگرێتەوە کە بریتین لە ئیلیمێنتێک واتا ناوی یان کڵاس یان id وەکو h1، h2 ناوی کەسی ڕاستەقینە لە HTML.
  • Universal Selector (سێلێکتەری گشتی): ستایل بە هەموو ئیلیمێنتەکان دەدات لە سەرپەڕەیەکی HTML هێماکەی {*} ناوەکی.
  • Attribute Selector (سێلێکتەری ئەتریبیوت): ستایل دەدات بە ئەتریبیوتی ئیلیمێنت وەکو src، href هێماکەی [].
  • ID Selector: هەڵیدەبژێرێت بە # لەگەڵ ناوێک دەبێت.
  • Class Selector: هەڵیدەبژێرێت بە (.) لەگەڵ ناوێک دەبێت.
  • گروپ بە سێلێکتەر: ستایل دەدات بە کۆمەڵێک لە ئیلیمێنتەکانی HTML پێکەوە وە لە ڕێگەی کۆماوە جیا دەکرێنەوە.
  • ئەتریبیوت سێلێکتەر: بەکاردێت بۆ دیاریکردنی ئەتریبیوتی ئیلیمێنتێک یان نرخی ئەتریبیوتە کە.
  • : سنگڵ کۆڵۆن: بەکاردێت بۆ دیاریکردنی سودۆ کڵاس (Pseudo-class).
  • :: دەبڵ کۆڵۆن: بەکاردێت بۆ دیاریکردنی سودۆ ئیلیمێنتی سێلێکتەر (Pseudo-element selector).

🎨 بەشی سێیەم: ڕەنگەکان (CSS Colors)

ڕەنگ پێدانی گرنگی بە وێب دیزاین دەگرێت پێدەدات کە شێوە پێدەری وێب ڕەنگە دەبێتین بۆ ئیلیمێنت.

شێوازەکانی پێدانی ڕەنگ:

  • Color Name: دەتوانین بۆ هەر ڕەنگێک ناوێک بەکاربهێنین.
  • RGB Format: بەکاردێت بۆ ناساندنی ڕەنگێک بۆ ئیلیمێنتێکی HTML بۆ دیاریکردنی نرخی RGB لە نێوان 0 to 255.
  • RGBA Format: بە هەمان شێوەی هەیکسا دەسیمێل دەستپێدەکات بە # وە لە شەش ژمارە پێکدێت لە نێوان 0 to f.

جۆرەکانی ڕەنگ لە CSS:

  • CSS بۆ ڕەنگە بەکاردێت بۆ دانانی چواردەوری ئیلیمێنتێک وە بۆ جیاکردنەوەی بەشێکی وێنەکەی مان.
  • بۆردە ستایل: بەکاردێت بۆ دیاریکردنی جۆری چوارچێوەکەمان وەکو سۆلید و داشێد و دۆتد و ...
  • بۆردە ڕوبەر: بەکاردێت بۆ دانانی بۆچوار چێوەکە.
  • بۆردە کەلەر: بەکاردێت بۆ دیاریکردنی ڕەنگی چوارچێوەکە.
  • بۆردە ڕادیەس: دروستکردنی قەوس یان کەوانەی بۆ سوچی چوارچێوەکە.
  • مارجین: بۆشاییەک دروستدەکات لە سەرو ئیلیمێنتێک لە دەرەوەی دیواری دەوری ئەو قەوبۆرتە.

📏 بەشی چوارەم: دیزاین و ڕێکخستنی لاپەڕە

لە وێب دیزاین مارجین ڕۆڵێکی گرنگی هەیە وەکو بۆشاییەک بە دەوری ئیلیمێنتێک دەناسێنێتەوە.

  • PX: پێوانەیی باوە کە ژمارەیەکی جێگیرە یاری دەکات بۆ PX کە.
  • ڕێژەی سەدی %: مارجین حیساب دەکرێت بە ڕێژەی سەدی کە درێژی و پانی ئیلیمێنتێک لە خۆدەگرێت.
  • سپەیس ئەیلیمنێت: جیاکردنەوەی ئیلیمێنتەکان بۆ ئەوەی باشتر بخوێنرێتەوە.
  • کریتینگ لەی ئاوت: بە ناساندنی کە بۆشاییەکان بۆ ڕێک خستنە وەی لەیئاوت.
  • ئەلاین ئیلیمێنتس: بەکارهێنانی مارجین بۆ ئەوەی ئیلیمێنتەکانمان بڵاو ببنەوە یان پاڵپشتی ستوونی یان ئاسۆیی.
  • ڕیسپۆن سیڤنێس: جێبەجێ کردنی میدیا کوێرییەکان بۆ ڕێک خستنی مارجین کان بۆ قەبارەی سکرینە جیاوازەکان یان شاشە جیاوازەکان.
  • ئاوت لاین: تایبەتمەندییەکی CSS ە کە بەکاردێت بۆ بەکارهێنانی هێڵێک بە دەوری ئیلیمێنتێکدا.
  • ئاوت لاین ویندز: بەکاردێت بۆ دیاریکردنی پانی چوارچێوەیەکی ئیلیمێنتێکی تایبەت.
  • ئاوت لاین ئۆفسێت: بەکاردێت بۆ دیاریکردنی بۆشایی نێوان بۆردەرەکان و ئاوت لاینەکان.

🔤 بەشی شەشەم: فۆنتەکان لە CSS (Fonts)

  • CSS font: بەکاردێت بۆ ستایل دان بە تێکست کە لە ناو ئیلیمێنتێکی HTML دا نووسرا بێت.
  • CSS font Style: کردنی تایبەتمەندییەکانی CSS فۆنت کە دەبێتە هۆی جوان کردن و باشتر خوێندنەوەی نووسینەکە لە ناو وێبپەیجەکەماندا.
  • CSS Font-weight: بەکاردێت بۆ تۆخ کردنەوەی فۆنت نووسین.
  • Font-size: بەکاردێت بۆ دیاریکردنی قەبارەی نووسین.
