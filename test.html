📦 بەشی دووەم: گۆڕاوەکان، کۆمێنت و سینتاکس (Variables)

گۆڕاو (Variable): گۆڕاوەکان پێکدێن لە پیتەکان Characters وە ژمارەکان Numbers.

❌ ڕێگەپێدراو نییە (Invalid):
  • ناتوانیت بە ژمارە دەست پێ بکات: 123
  • ناکرێت هێماکانی تێدابێت: Test! , Name%
  • ناکرێت ئەم وشانە بەکاربهێنیت بۆ ناوی گۆڕاو: for, if, while, import
✅ ڕێگەپێدراوە (Valid):
  • دەتوانیت بە ئەندەرسکۆر دەست پێ بکات: _name
  • دەکرێت پیتی گەورە و بچووک بێت: AGE , aGE
  • نموونەی تر: Name1 , a1111 , my_name

تێبینییە گرنگەکانی سینتاکس:

  • کۆمێنت: هاش تاگ # بەکاردێت بۆ کۆمێنت (تێبینی نووسین لەناو کۆددا).
  • بۆشاییەکان (Indentation): Indentation in Python is meaningful. بەپێچەوانەی زمانەکانی تردا سپەیس مانادار نییە، بەڵام لە پایتۆن مانادارە و بۆ جیاکردنەوەی بلۆکەکان بەکاردێت.

🗂️ بەشی سێیەم: جۆرەکانی داتا (Data Types) و ژمارەکان

پشکنینی جۆری داتا بە type:
  • type(name) == str (T)
  • type(age) == int (T)
  • type(age) == str (F)

گۆڕینی داتا (Casting): بەکاردێت بۆ گۆڕینی داتاتایپێک بۆ یەکێکی تر.

جۆرەکانی داتاتایپ بریتین لە:

Complex - bool - list - Tuple - range - dict - set

ژمارەکان لە پایتۆن (Numbers):

سێ جۆری ژمارەمان هەیە:

  • int (Integer): ژمارەی تەواو نیشان دەدات بەبێ فاریزە (پۆینت).
  • float (Floating Point): بەکاردێت بۆ ژمارەی ڕێژەیی (پۆینتدار).
  • complex (Complex Number): ژمارە ئاڵۆزەکان، کە پێکدێت لە دوو بەش: بەشی ڕاستی وە بەشی خەیاڵی. 
    # بەشی خەیاڵی (imag)
Complex_number = 2 + 3j

# بەشی ڕاستی (real) بە فەنکشن
Complex_number = complex(2, 3)

➕ بەشی چوارەم: ئۆپەرەیتەرەکان (Operators)

١. Arithmetic Operators (کردارە بیرکارییەکان)

ئاماژە دەگرێتەوە بۆ: + , - , * , / , % , ** , //

+ is also used to concatenate string values: هەروەها نیشانەی کۆکردنەوە بۆ لکاندنی (پێکەوەنانی) دوو سترینگیش بەکاردێت.

٢. Assignment Operators (یەکسان کردن)

بەکاردێت بۆ یەکسان کردنی نرخێک بە گۆڕاوێک.
وەک: = , += , -= , *= , /= , %= (نموونە: a = 4)

٣. Comparison Operators (بەراوردکارەکان)

بۆ بەراوردکردن بەکاردێن: == , != , > , < , >= , <= (نموونە: a == 2)

٤. Logical Operators (باری لۆژیکی / Boolean)

  • and: ئەگەر هەردووکیان True بوو ئەنجام True دەبێت.
  • or: ئەگەر هەردووکیان False بوو ئەنجام False دەبێت (واتە ئەگەر یەکێکیان True بێت، ئەنجام True یە).
  • not: پێچەوانەی دەکاتەوە.

٥. Bitwise Operators

  • & ➔ And
  • | ➔ OR
  • ^ ➔ XOR
  • ~ ➔ NOT
  • << ➔ Shift left
  • >> ➔ Shift Right

٦. Identity و Membership ئۆپەرەیتەرەکان

  • is : پێی دەوترێت Identity operator. بەکاردێت بۆ بەراوردکردنی دوو ئۆبجێکت.
  • in : پێی دەوترێت Membership operator. دەپشکنێت بزانێت قاڵبێک لەناو لیستێکدا هەیە یان نا.

٧. Ternary Operator

سوودمەندە بۆ ئاسانکاری مەرجەکان (دوو شت بەیەکەوە دەبەسترێتەوە).

🔤 بەشی پێنجەم: سترینگ و فەنکشنەکانی (String Methods)

تێبینی: سترینگ دەتوانرێت بە جووت کۆتەیشن یان سینگڵ کۆتەیشن بنووسرێت بەپێی سینتاکسی تایبەت. وە str() بەکاردێت بۆ گۆڕینی ژمارە بۆ سترینگ.

میتۆدەکانی ناو سترینگ:

  • lower(): گۆڕین بۆ سمۆڵ.
  • islower(): چێک کردنی سمۆڵ (بزانێت سمۆڵە؟).
  • upper(): گۆڕین بۆ کەپیتاڵ.
  • isupper(): چێک کردنی کەپیتاڵ.
  • startswith(): چێک دەکات ئایا سترینگ بەمە دەست پێدەکات.
  • split(): دابەشکردنی یان بڕینی سترینگ.
  • len(): بۆ حیساب کردنی درێژی سترینگ (ژمارەی پیتەکان).

نیشانەکانی ناو سترینگ (Escape Characters):

  • \ ➔ بەکاردێت بۆ سڕینەوەی کاریگەری نیشانەکان (Escape).
  • \" ➔ بۆ بانگکردنی دەبڵ کۆتەیشن لە ناو سترینگدا.
  • \t یان \b ➔ بۆ سپەیس و تاب.

✅ بەشی شەشەم: یاساکانی True و False و فەنکشنەکان

یاساکانی باری لۆژیکی:
  • سفر (0): هەمیشە فۆڵسە (False).
  • ڕەقەمەکان: هەموو ژمارەکانی تر تەنانەت ژمارە سالبەکانیش ترون (True).
  • سترینگ: سترینگ تەنها کاتێک فۆڵسە (False) کە بەتاڵ بێت واتا هیچی تێدا نەبێت "".

فەنکشنە پشکنەرەکان:

  • isinstance(): فەنکشنێکە دەگەڕێتەوە کە ئایا دروستە/یەکێکە لە جۆرێکی دیاریکراو.
  • all(): فەنکشنی all تروو (True) دەگەڕێنێتەوە ئەگەر هەموو ئەندامەکانی ناوەکەی تروو بن (واتە هەمووی دەبێ تروو بێت).

🛑 بەشی حەوتەم: کۆنستانت، ئینوم و ئینپوت (Constants, Enums, User Input)

کۆنستانتەکان (Constants)

  • هیچ ڕێگەیەک نییە کە بتوانیت بەکاربێنیت بۆ کۆنستانت لە پایتۆن. (T/F)
  • بە هیچ جۆرێک ناتوانیت دووبارە نرخ بدەیتەوە بە کۆنستانت.

ئینومەکان (Enums)

لە شوێنی ئەوەی کۆنستانت بەکاربهێنیت، لە پایتۆن دەتوانیت Enum بەکاربهێنیت. بۆ بەکارهێنانی دەبێت ئیمپورتی بکەین:

from enum import Enum

# دەتوانین بەم شێوازانە بەکاری بهێنین:
print(State(1))
State.Active
State['ACTIVE']

ئینپوت (User Input)

لە ناو پایتۆن دەتوانیت کۆماندێکی تایبەت بەکاربهێنیت بۆ وەرگرتنی زانیاری لە بەکارهێنەر، بە بەکارهێنانی فەنکشنی input(). دەتوانیت داتایەک دابنێیت لە ڕێگەی بەکارهێنەرەوە بە بەکارهێنانی ئەم فەنکشنە.

🔀 بەشی هەشتەم: کۆنترۆڵ ستەیتمێنت و لیستەکان (Control Statements & Lists)

کۆنترۆڵ ستەیتمێنت (Control Statements)

  • کاتێک مەرجێک دادەنێیت، ئەوا ئەو Block ی مەرجەکە جێبەجێ دەبێت.
  • بلۆک (Block) چییە؟ بەشێکە لە ڕیزبەندییەک و جیا دەکرێتەوە بۆ دروستکردنی مانایەک. دەتوانیت جێبەجێی بکەیت لە یەک کاتدا یان دەتوانیت دواتر جێبەجێی بکەیت.
  • else block: جێبەجێ دەبێت لە دوای if ئەگەر مەرجی if فۆڵس (False) بێت.
  • elif: دەتوانیت بەکاری بهێنیت لە نێواندا وەکو جێگرەوەیەک بۆ ئەو کاتەی کە if جێبەجێ نەبوو، ئەم مەرجە جێبەجێ ببێت.

لیستەکان (Lists)

  • ستراکتچەرێکی داتای تایبەت بە خۆی هەیە لە ناو پایتۆندا.
  • لە چوارچێوەی کەوانەیەکی شکاوەدا (کەوانەی چوارگۆشە []) ڕێکدەخرێت و کۆمەڵێک داتای جیاواز لەخۆ دەگرێت.
  • لیست دەتوانێت کۆمەڵێک ڤالوی جیاواز لەخۆ بگرێت. لە ڕێگەی کۆماوە جیا دەکرێنەوە، وە لە ناو لیستدا هەموو جۆرەکانی داتا دەکرێت هەبن یان نەبن.
  • Index: دۆزینەوەی ئیندێکسی کاڵاکان یان ئایتمەکان لە ڕێگەی میتۆدی index دەکرێت (ئەگەر ئایتمێک لە ناو لیستدا نەبێت ئێررۆر دەدات).
  • Slices: بە مانای وەرگرتنی بەشێک لە لیستەکەمان لە ڕێگەی :.
  • len(): ژمارەی ئایتمەکان لە لیستەکەمان بۆ دەدۆزێتەوە.
  • append(): دەتوانین میتۆدی ئەپێند بەکاربهێنین بۆ زیادکردنی ئایتمێک بۆ کۆتایی لیستەکەمان (نموونە: items.append("Test")).
  • extend(): بە هەمان شێوەی ئەپێند بۆ زیادکردنە، وە += ئۆپەرەیتەریش بۆ هەمان مەبەست بەکاردێت.
  • remove(): فەنکشنی ڕیموڤ بەکاردێت بۆ سڕینەوەی ئایتمێک.
  • insert(): زیادکردنی ئایتمێک لە ناوەڕاستی لیستەکەماندا (بە پێدانی شوێنەکەی).
  • sort(): دەتوانین لیستەکانمان ڕیزبکەین لە ڕێگەی میتۆدی سۆرت.

📦 بەشی نۆیەم: توپل، دیکشنەری، و سێت (Tuples, Dictionaries, Sets)

توپلەکان (Tuples)

  • پایتۆن وەکو گروپێک دایان دەنێت کە ڕێگە نادات دەستکاری بکرێت.
  • لە توپلەکاندا (کە پێچەوانەی لیستن) دەتوانیت توپل دروست بکەیت، بەڵام ناتوانیت هیچ داتایەکی لێ بسڕیتەوە یان زیادی بکەیت.
  • بە هەمان شێوەی لیست دروست دەکرێت، بەڵام لە جیاتی کەوانەی چوارگۆشە []، کەوانەی ئاسایی (بازنەیی) () بەکاردێت.
تێبینی: دەتوانیت توپلێکی تازە دروست بکەیت لە شێوەی زیادکردن وە بەکارهێنانی ئۆپەرەیتەری +. وەکو توپلێکی تازە وایە نەک کۆنەکە زیاد بکرێت.

دیکشنەری (Dictionaries)

  • کاتێک لیستێکمان هەبێت کە کۆمەڵێک ڤالو بە یەکەوە دەبەستێتەوە، پێی دەوترێت دیکشنەری کە کۆمەڵێک کی و ڤالو (Key-Value) لەخۆ دەگرێت.
  • ناتوانین Key بگۆڕین بەڵام دەتوانین ڤالوەکە بگۆڕین.
  • get(): میتۆدێکە دەگەڕێت بۆ داتایەک بەپێی کییەکەی.
  • pop(): ڕیموڤ میتۆدە بۆ سڕینەوەی داتایەک لەناو دیکشنەری.
  • popitem(): بۆ سڕینەوەی کۆتا Key-Value لەناو دیکشنەرییەکە.
  • in: لە ڕێگەی in چێک دەکەیتەوە کە ئایا هەیە یان نا.
  • keys(): میتۆدی کییز بەکاردێت بۆ هێنانەوەی هەموو کییەکانی ناو دیکشنەرییەکە.
  • values(): بۆ هێنانەوەی هەموو ڤالوەکانی ناو دیکشنەرییەکە بەکاردێت.
  • items(): بۆ وەرگرتنەوەی هەردووکیان پێکەوە (Keys و Values) بەکاردێت.
  • len(): بەکاردێت بۆ وەرگرتنەوەی درێژی تێکستەکە یان سترینگەکە، یان ئایتمەکانی ناو لیستەکان و دیکشنەرییەکان.
  • بۆ زیادکردنی Value و Key: لە ڕێگەی ناوی دیکشنەرییەکەوە، بۆ نموونە dog['fav food'] = 'meat'.
  • del: بۆ سڕینەوەی Key یەک لە دیکشنەریدا بەکاردێت.
  • copy(): بۆ لەبەرگرتنەوە یان کۆپی کردنی دیکشنەرییەکە.

سێتەکان (Sets)

  • سێت جۆرێکی ترە لە کۆلێکشن کە ڕێگە نادات داتای دووبارەی تێدا بێت. (T/F)
  • گرنگترین تایبەتمەندی سێتەکان ئەوەیە کە داتای دووبارە قبوڵ ناکات.
  • وەکو دیکشنەری دەنووسرێت بەڵام Key ی نییە.
  • سێتێکی تایبەتیش هەیە کە ناتوانرێت گۆڕانکاری تێدا بکرێت کە پێی دەوترێت frozenset.
  • سینتاکسی سێت: names = {"Roger", "Syd"} کە لە جیاتی کەوانەی چوارگۆشە یان ئاسایی، کەوانەی لول (Curly Braces) بەکاردێت.
  • union() یان |: هەموو داتاکانی نێوان هەردوو سێتەکە دەخاتە ناو یەکەوە (بەبێ دووبارەبوونەوە). (نموونە: set1 | set2 یان set1.union(set2)).
  • difference() یان -: جیاوازییەکانی نێوان دوو سێت دەدۆزێتەوە.

⚙️ بەشی دەیەم: فەنکشنەکان (Functions)

  • دروستکردنی کۆمەڵێک کۆدە کە دەتوانین چەندین جار بانگی بکەین (بەکاری بهێنینەوە).
  • هیچ فەنکشنێک دروست نابێت بەبێ دەستپێکردن بە وشەی def، وە دواتر ناوی فەنکشنەکە و پاشان کەوانە () و لە کۆتاییدا دوو خاڵ : دێت.
  • دەتوانین زیاتر لە جارێک Function بانگ بکەینەوە بە پێدانی ناوەکەی.
  • لە Function دا دەتوانیت پارامێتەرت هەبێت. پارامێتەرەکان وەکو گۆڕاو وان لەناو فەنکشنەکەدا و کاتی بانگکردنەوەی دەتوانیت داتایان پێ بدەیت.
  • پارامێتەرەکان وەکو ڕیفرێنسن و کاتی بانگکردنەوە دەبێت هەموو جۆرەکانی لەناو کەوانەدا بن، بەڵام دەکرێت گۆڕاوەکان لە دەرەوەی فەنکشنیشدابن و بانگ بکرێنەوە.
  • return: دەتوانین نرخێک بگەڕێنینەوە لە ڕێگەی ووشەی ڕیتێرن. وە وشەی ڕیتێرن بەکاردێت لە کۆتایی فەنکشنەوە.
